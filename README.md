# ETH Zürich CS Summaries
My summaries and cheat sheets for CS BSc courses at ETHZ.

These notes were designed for my own use, so they may abbreviate or omit certain (sub-)topics.

If you notice any factual inaccuracies, feel free to open a pull request.

# Use
The notes are written in [Typst](https://typst.app/). Use `typst compile FILENAME.typ` to compile them.
